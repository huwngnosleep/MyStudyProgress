một hệ phụ thuộc tuyến tính (linearly dependent) là hệ mà có một vector
biểu diễn tuyến tính theo các vector còn lại

thì khi đó combination của hệ có add thêm bao nhiêu cái vector phụ thuộc vào thì vẫn không thể tạo ra chiều mới

- kiểm tra xem hệ có là independent hay không
nếu là phụ thuộc (dependent) => v1 = c2v2 + c3v3 + ... + cnvn
chuyển vế => -v1 + c2v2 + c3v3 + ... + cnvn = 0

như vậy mới sinh ra cách để kiểm tra hệ có là độc lập tuyến tính hay không bằng phương trình
c1v1 + c2v2 + ... + cnv2 = 0

nếu tất cả các C = 0 => phụ thuộc
các C != 0 => độc lập
